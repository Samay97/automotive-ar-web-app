## Development setup

Install the Angular CLI globally:

```
npm install -g @angular/cli
```


Install dependencies:

```
cd automotive-ar-web-app
npm install
```


Run the application:

```
npm start
```


Or run the application on host:

```
npm run start:host
```

Navigate to `http://localhost:4200/`
